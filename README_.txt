Instalacja środowisk:

- Microsoft SQL Server wraz z Microsoft SQL Server Management Studio oraz dodatku SQL Server Agent

- Microsoft Visual Studio 2019

Utworzenie bazy danych 

Należy uruchomić Microsoft SQL Server.

Z użyciem Microsoft SQL Server Management Studio przywrócić bazę danych z pliku bank.bak.

Następnie należy uruchomić SQL Agent oraz wykonywać przy pomocy SSMS wszystkie skrypty z folderu zaplanowane_zadania

Aplikację uruchamiamy przy użyciu Visual Studio 2019. Aby to zrobić należy uruchomić aplikację przy użyciu zawartego w folderze WebApplication1 pliku WebApplication1.sln
